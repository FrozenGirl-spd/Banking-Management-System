# Banking-Management-System

## Using C++ 

Features:
1. Login
  a. Bank Manager
    i. View all users
    ii. Increase/Decrease Rate of Interest
    iii. Open/Close accounts
  b. User
    i. Opening/Close your account
    ii. Show account info with balance using simple interest
    iii. Deposit
    iv. Withdraw
